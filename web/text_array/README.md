# Кастомный интерфейс для LoTextArray

Этот кастомный интерфейс предоставляет удобный способ работы с массивом текстов через закладки.

## Функциональность

- **Закладки**: Каждый текст отображается в отдельной закладке
- **Добавление/удаление**: Можно добавлять новые тексты и удалять существующие
- **Сворачивание**: Интерфейс можно сворачивать для экономии места
- **Редактирование**: Каждый текст можно редактировать в отдельном текстовом поле

## Использование

1. Добавьте узел "Lo:TextArray" в ваш workflow
2. Нажмите на заголовок "Массив текстов" для разворачивания интерфейса
3. Используйте кнопку "+ Добавить текст" для создания новых закладок
4. Кликайте по закладкам для переключения между текстами
5. Используйте кнопку "×" на закладке для удаления текста

## Технические детали

- Интерфейс создан с использованием чистого JavaScript
- Стилизация выполнена через CSS
- Данные автоматически синхронизируются с узлом ComfyUI
- Поддерживается неограниченное количество текстов
